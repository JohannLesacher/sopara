# Projet Sage 11 Default

## Références
- URL locale : https://sage11default.test
- Thème : `web/app/themes/default/`

## Stack
Ce projet utilise la stack Roots/Sage 11. Le skill `sage-11` est installé
et contient toute la référence technique — s'y référer pour les patterns,
conventions et exemples de code.

## Post types actifs
- `evenement` — concerts/événements

## Plugins actifs notables
- MetaBox + MB Blocks

## Conventions propres au projet
- Palette de couleurs : voir `theme.json`
- Catégorie de blocs custom : `sur-mesure`
- Namespace PHP : `App\`

## Ce qu'il ne faut pas toucher
- `public/build/` — généré par Vite
- `functions.php` — bootstrap uniquement
